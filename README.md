# 🎮 텍스트 RPG 세계관에서 하렘을 구축하는 방법
> **"정신을 차려보니 텍스트 속 세상... 게다가 내 동료(하렘)들이 전부 적이었다?!"**
> 모태솔로 개발자들이 모여 만드는 스릴 만점 현대 판타지 미연시 RPG 프로젝트.

---

## 🛠️ 프로젝트 개요
* **프로젝트 명** : 텍스트 RPG 세계관에서 하렘을 구축하는 방법
* **팀 이름** : 모태쏠로팀
* **장르** : 미연시 (미소녀 연애 시뮬레이션) RPG
* **시대상** : 현대 판타지
* **핵심 테마** : 
  * ⚔️ **적들을 동료로!** (매력적인 적 여간부/몬스터들을 공략하여 아군 하렘으로 영입)
  * 🔄 **속성 상성 관계** (불, 물, 풀 등의 상성을 이용한 전략적 전투와 밀당)
  * 👸 ** 개성 넘치는 캐릭터** (핫걸, 냉미녀, 초식녀, 그리고 평범미녀까지!)

---

* **[문승종] 팀장** : 팀원 감시 (※ 사다리타기로 공정하게 결정됨)

### 👑 운영 및 기획 라인
* **[문승종] 팀장** : 팀원 감시 + **[5. 아이템 시스템]** 개발
* **[조성운] 부팀장** : 팀장 보조 및 실제 실무/팀 관리 총괄 + **[2. 적(Enemy) 시스템]** 개발
* **[김희진] 기획팀장** : 프로젝트 깃(Git) 관리 및 파트너 조율 + **[6. 게임로그확인 시스템]** 개발

### 🎭 분위기 및 서포트 라인
* **[성태현] 예능팀장** : 타 팀 정탐 및 정보 수집 + 열심히 춤추기 + 찰떡같은 BGM 탐색 + 팀 분위기 메이커 + **[4. 레벨업 시스템]** 개발
* **[임윤택] 발표팀장** : 프로젝트의 얼굴, 발표 및 발표 자료 제작 총괄 + **[3. 전투 시스템]** 개발
* **[임우현] 청소부장** : 깔끔한 노션(Notion) 페이지 개설 및 핵심 회의록 관리 + **[1. 플레이어 캐릭터]** 개발

---

## 📝 구현 시스템 명세 (System Specification)

1. **[우현] 플레이어 캐릭터 (Player)**
   * 현대 판타지 세계관에 뛰어든 주인공의 스탯, 성향, 호감도 센서 구현.
2. **[성운] 적 (Enemy)**
   * 주인공의 앞길을 막아서는 매력적인 적 캐릭터 및 몬스터들 배치.
3. **[윤택] 전투 시스템 (Battle)**
   * 텍스트 기반의 턴제 전투. 속성 상성을 활용한 공략 및 영입(설득) 시스템의 핵심.
4. **[태현] 레벨업 (Level Up)**
   * 전투 승리 및 히로인과의 유대감 상승에 따른 능력치 성장 구조 설계.
5. **[승종] 아이템 (Item)**
   * 전투용 장비부터 미녀들의 마음을 사로잡을 비밀스러운 선물 아이템 구현.
6. **[희진] 게임로그확인 (Game Log)**
   * 지나간 전투 기록은 물론, 히로인들과의 짜릿한 대사 기록을 다시 볼 수 있는 비록 기능.

---

## 👩‍❤️‍👨 영입 가능한 주요 히로인 속성
* 🔥 **핫걸 (불 속성)** : 화끈한 성격과 화력으로 전장과 주인공의 마음을 태우는 그녀.
* ❄️ **냉미녀 (물 속성)** : 얼음처럼 차갑지만 호감도를 채우면 누구보다 달달해지는 그녀.
* 🌿 **초식녀 (풀 속성)** : 평화주의자 같지만 알 수 없는 매력으로 사람을 끌어당기는 그녀.
* ✨ **미녀 (평범 속성)** : 평범한 게 최고! 클리셰를 파괴하는 가장 완벽한 정통파 미녀.
