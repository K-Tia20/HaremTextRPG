# 🎮 텍스트 RPG 세계관에서 하렘을 구축하는 방법
> 모태솔로 개발자들이 모여 만드는 스릴 만점 현대 판타지 미연시 RPG 프로젝트.
> 
---

## 🛠️ 프로젝트 개요
* **프로젝트 명** : 텍스트 RPG 세계관에서 하렘을 구축하는 방법
* **팀 이름** : 모태솔로
* **장르** : 미연시 (미친 연애 시뮬레이션) RPG *미소녀 연애 시뮬레이션 아닙니다
* **시대상** : 현대 판타지
* **핵심 테마** : 
  * ⚔️ **적들을 동료로!** (매력적인 히로인들을 정복하여 아군 하렘으로 영입)
  * 🔄 **속성 상성 관계** (핫걸, 냉미녀, 초식녀 등의 성격 상성을 이용한 전략적 전투와 밀당)
  * 👸 ** 개성 넘치는 캐릭터** (어마어마한 도트 이미지)

---

### 👑 팀장만 5명
* **[문승종] 팀장** : 팀원 감시 (※ 아이씨유~!) + **[5. 아이템 시스템]+[1. 플레이어 캐릭터]** 개발

* **[조성운] 부팀장** : 팀장 보조 및 실제 실무/팀 관리 총괄 + **[2. 적(Enemy) 시스템]** 개발
* 
* **[김희진] 기획팀장** : 프로젝트 깃(Git) 관리 및 파트너 조율 + **[6. 게임로그확인 시스템]** 개발

* **[성태현] 예능팀장** : 타 팀 정탐 및 정보 수집 + 열심히 춤추기 + 찰떡같은 BGM 탐색 + 팀 분위기 메이커 + **[4. 레벨업 시스템]** 개발

* **[임윤택] 발표팀장** : 프로젝트의 얼굴, 발표 및 발표 자료 제작 총괄 + **[3. 전투 시스템]** 개발

### 🎭 부장 1명(UI 셔틀)

* **[임우현] 청소부장** : 깔끔한 노션(Notion) 페이지 개설 및 핵심 회의록 관리 + **[UI 시스템]** 개발

## 👩‍❤️‍👨 영입 가능한 주요 히로인 속성
* 🔥 **핫걸** : 화끈한 성격과 화력으로 전장과 주인공의 마음을 태우는 그녀.
* ❄️ **냉미녀** : 얼음처럼 차갑지만 호감도를 채우면 누구보다 달달해지는 그녀.
* 🌿 **초식녀** : 평화주의자 같지만 알 수 없는 매력으로 사람을 끌어당기는 그녀.
* ✨ **평범녀** : 평범한 게 최고! 클리셰를 파괴하는 가장 완벽한 정통파 미녀.
