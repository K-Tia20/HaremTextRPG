[COMMIT MESSAGE]

Subject: 시스템 아키텍처 하이브리드 통합 및 UI 물리 안정성 강화 (Dead Zone 설계)

본 커밋은 팀원들의 최신 풀(Pull) 이후 발생한 대규모 빌드 에러를 해결하고, 
사용자 기획 의도에 맞춘 시네마틱 UI 프레임워크를 완벽하게 재이식한 결과물입니다.

[1. 아키텍처 및 시스템 통합]
- 하이브리드 월드 관리: 팀의 shared_ptr 기반 동적 생성 방식과 기존 싱글톤 접근법을 동시에 지원하는 유연한 구조로 World 클래스 재건.
- 출력 스트림 단일화: Alba, Shop, City, Battle 등 모든 로직의 std::cout을 UI 엔진의 PrintLog로 우회시켜 레이아웃 보호.
- 메인 루프 교정: 구역 활동 종료 후 항상 '내 방(BG_Room)' 배경과 거점 메뉴가 출력되도록 씬 매니지먼트 복구.

[2. UI/UX 및 물리 안정성 (핵심)]
- 물리적 데드존(Dead Zone) 설계: 콘솔 44번 행을 봉인하고 모든 UI 하단을 43번 행으로 클램핑하여 엔터 연타 시의 화면 밀림 현상을 원천 차단.
- 입력 레이어 격리: 모든 사용자 입력을 스마트폰 하단 전용 구역으로 집중시키고, 입력 즉시 잔상을 제거하는 연출 적용.
- 시네마틱 오프닝 재가동: 프롤로그 타이핑 -> 파란 트랜지션 -> UI 조립 시퀀스 1회 한정 실행 플래그 도입.

[3. 로직 보강 및 버그 수정]
- 평범녀 루트 구현: 누락되었던 SetNormalGirl 연출(이미지 및 전용 대사) 추가.
- 배틀 시스템 안정화: 전투 중 아이템 사용 시 발생하던 NULL 포인터 역참조(Access Violation) 방지 로직 및 예외 처리 강화.
- 경고 및 인코딩 정화: C++20 표준 상향, /utf-8 플래그 적용, Defs.h 등 주요 헤더 UTF-8 BOM 변환으로 빌드 경고 1200여 개 제거.

[4. 협업 준비]
- 사운드 엔진 구축: BGM 작업자를 위한 MCI 기반 CSoundManager 및 SOUND_GUIDE.txt 생성.
- 코드 후크 배치: 로직 곳곳에 사운드 삽입을 위한 // [SOUND] 주석 포인트 확보.

이로써 팀원들의 논리(Logic)와 사용자님의 표현(UI)이 기술적 충돌 없이 공존하는 최종 안정화 버전을 완성함.
